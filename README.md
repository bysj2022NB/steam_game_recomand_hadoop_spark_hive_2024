## 流程： 

1.Python爬取steam或者其他游戏站点数据存到.csv和mysql数据库(双备份)；

2.将.csv文件上传到hdfs文件系统；

3.使用hive建表建库，并且导入.csv文件建立数据仓库;

4.一半分析指标使用hive_sql完成，一半分析指标使用Spark完成；

5.分析结果使用sqoop导入mysql数据库；

6.使用Flask+echarts搭建可视化大屏；
